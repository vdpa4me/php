수업 방법 변경 공지


1. 향후 수업은 영어식 문장 구조를 활용한 한글->영어 변환 훈련만 진행할 것입니다.
2. 수업은 매주 20~30분간 진행될 것입니다.
3. 수업 전반부에는 지난주 배운 5문장을 테스트 합니다. ( OLV : Obtained Level)
4. 수업 후반부에는 새로운 5문장을 학습합니다. (CLV : Class Level)
5. OLV는 테스트에 합격하면 OLV가 한단계 상승하며 200 Gold가 지급 됩니다.
6. 녹음 숙제는 더이상 없습니다. 수업 후 다음 수업 전까지, 새로 배운 5문장을 암기하시면 됩니다.
